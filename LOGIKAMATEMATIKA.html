Logika Matematika

Mempelajari dasar logika biner (0 dan 1), tabel kebenaran, serta kalimat logika dan simbol-simbol formal dalam matematika diskrit.

Logika Biner (0 dan 1)

Dalam logika matematika, nilai kebenaran hanya memiliki dua kemungkinan: 1 (Benar) dan 0 (Salah). Konsep ini menjadi dasar sistem digital dan pemrograman. 


1 = Benar (True)
0 = Salah (False)

Contoh:
Jika P = 1 dan Q = 0

Tabel Kebenaran

Tabel kebenaran digunakan untuk menentukan hasil operasi logika seperti AND (∧), OR (∨), dan NOT (¬). 


P   Q   P ∧ Q
1   1     1
1   0     0
0   1     0
0   0     0

Kalimat & Simbol Logika

Logika matematika mempelajari pernyataan (proposisi) seperti kalimat ajakan, bantahan, implikasi, dan biimplikasi menggunakan simbol formal. 


¬P   : Negasi (Tidak P)
P ∧ Q : Konjungsi (Dan)
P ∨ Q : Disjungsi (Atau)
P → Q : Implikasi (Jika... maka...)
P ↔ Q : Biimplikasi

Contoh:
Jika hujan maka jalan basah
H → B
