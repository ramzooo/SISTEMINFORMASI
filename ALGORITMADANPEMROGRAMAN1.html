Algoritma dan Pemrograman 1

Mempelajari konsep dasar algoritma, logika pemrograman, struktur data sederhana, serta implementasi menggunakan bahasa C++ dalam era digital.

Konsep Dasar Algoritma

Algoritma adalah langkah-langkah sistematis dan logis untuk menyelesaikan suatu masalah. Dalam dunia digital, algoritma digunakan untuk memproses data, menghitung, mengurutkan, dan mengambil keputusan secara otomatis. 


Contoh Algoritma Sederhana:
1. Mulai
2. Masukkan angka
3. Jika angka > 0 tampilkan "Positif"
4. Selesai

Dasar Pemrograman C++

C++ adalah bahasa pemrograman yang digunakan untuk mengimplementasikan algoritma ke dalam bentuk kode yang dapat dijalankan oleh komputer. Dalam C++ kita belajar tentang variabel, tipe data, input-output, percabangan, dan perulangan. 


#include <iostream>
using namespace std;

int main() {
    string nama;
    cout << "Masukkan nama: ";
    cin >> nama;
    cout << "Halo " << nama;
    return 0;
}

Struktur Dasar C++

Dalam pembelajaran C++, kita memahami struktur program seperti: deklarasi variabel, operator, percabangan (if-else), perulangan (for, while), serta fungsi. 


// Percabangan
if (nilai >= 75) {
    cout << "Lulus";
} else {
    cout << "Tidak Lulus";
}

// Perulangan
for(int i = 1; i <= 5; i++){
    cout << i << endl;
}

Struktur Data Dasar

Struktur data digunakan untuk menyimpan dan mengelola data dalam program. Pada tahap awal, mahasiswa mempelajari array dan pengolahan data sederhana. 


// Array dalam C++
int angka[5] = {1, 2, 3, 4, 5};

for(int i = 0; i < 5; i++){
    cout << angka[i] << endl;
}
