Pemrograman Web 1

Mempelajari dasar HTML, CSS, dan JavaScript sebagai fondasi membangun website modern dan interaktif.

HTML (Struktur Website)

HTML digunakan untuk membangun struktur halaman web seperti heading, paragraf, link, gambar, dan form. 


<h1>Judul</h1>
<p>Ini paragraf</p>
<a href="#">Klik</a>

CSS (Desain & Layout)

CSS digunakan untuk mengatur warna, jarak, ukuran, dan tata letak halaman. 


body {
    background: #f4f4f4;
}

h1 {
    color: blue;
}

JavaScript (Interaktivitas)

JavaScript membuat website menjadi dinamis dan interaktif. 


let nama = "jojo";
alert("Halo " + nama);

if(nama === "jojo"){
    console.log("Mahasiswa Sistem Informasi");
}
