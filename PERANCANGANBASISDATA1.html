Perancangan Basis Data I

Mempelajari konsep dasar database, model relasional, serta perancangan struktur tabel yang efisien dan terorganisir.

Pengertian Database

Database adalah kumpulan data yang tersimpan secara terstruktur dan saling berhubungan sehingga mudah untuk diakses, dikelola, dan diperbarui. 


Contoh:
Data Mahasiswa
- NIM
- Nama
- Jurusan
- Semester

Sistem Manajemen Basis Data (DBMS)

DBMS adalah perangkat lunak yang digunakan untuk mengelola database. Salah satu DBMS yang populer adalah MySQL. 


Fungsi DBMS:
- Membuat database
- Mengelola tabel
- Mengatur relasi
- Menjalankan query

Bahasa SQL

SQL (Structured Query Language) adalah bahasa yang digunakan untuk mengelola dan memanipulasi data dalam database. 


SELECT * FROM mahasiswa;
INSERT INTO mahasiswa VALUES (...);
UPDATE mahasiswa SET nama='jojo';
DELETE FROM mahasiswa WHERE id=1;
